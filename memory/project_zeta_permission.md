Zeta AI로부터 ego-bloom 프로젝트의 API 데이터 사용을 공식 허락받음.

**허락 조건:**
- 개발자 포트폴리오로 사용하지 않을 것
- Zeta AI와 경쟁하는 서비스로 운영하지 않을 것
- 광고를 달아 수익을 내지 않을 것

**Why:** 공식 API 키 없이 비공개 엔드포인트를 호출하는 구조이지만 허가를 받았으므로 ToS 위반 걱정 없음.

**How to apply:** API 호출 방식이나 크롤링 빈도에 대해 과도하게 우려할 필요 없음. 단, 허가 조건(광고 수익화, 경쟁 서비스화, 포트폴리오 사용)을 벗어나는 방향의 기능 제안은 피할 것.
