# Ego Bloom — 제타 제작자 에고 발산기
## ELO 점수 체계

```
ELO = (총 대화량 × 3.0) + (팔로워 × 300) + (Top 20 캐릭터 대화량 × 0.5) + (캐릭터 평균 대화량 × 20.0) + (음성 재생 수 × 100.0)
```

### 등급 기준

| 티어 | 최소 점수 |
|---|---|
| Bronze | 0 |
| Silver | 50,000 |
| Gold | 225,000 |
| Platinum | 1,000,000 |
| Diamond | 4,500,000 |
| Master | 20,000,000 |
| Champion | 90,000,000 |

각 티어는 4단계(4→3→2→1)로 세분화되며, 진행률 바가 다음 단계까지의 진척도를 표시합니다.
